GESTimeOverlayClip

A GESSourceClip that overlays timing information on top.

Asset

The default asset ID is "time-overlay" (of type GES_TYPE_SOURCE_CLIP), but the framerate and video size can be overridden using an ID of the form:

time-overlay, framerate=60/1, width=1920, height=1080, max-duration=5.0
