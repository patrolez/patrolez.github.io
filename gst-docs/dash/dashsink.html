dashsink

Dynamic Adaptive Streaming over HTTP sink/server

Example launch line

 gst-launch-1.0 dashsink name=dashsink audiotestsrc is-live=true ! avenc_aac ! dashsink.audio_0 videotestsrc is-live=true ! x264enc ! dashsink.video_0
