directsoundsrc

Reads audio data using the DirectSound API.

Example pipelines

 gst-launch-1.0 -v directsoundsrc ! audioconvert ! vorbisenc ! oggmux ! filesink location=dsound.ogg

Record from DirectSound and encode to Ogg/Vorbis.
