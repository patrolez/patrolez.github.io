GstVideoTileInfo

Description of a tile. This structure allow to describe arbitrary tile dimensions and sizes.

Members

width (guint) –

The width in pixels of a tile. This value can be zero if the number of pixels per line is not an integer value.

height (guint) –
No description available
stride (guint) –

The stride (in bytes) of a tile line. Regardless if the tile have sub-tiles this stride multiplied by the height should be equal to size. This value is used to translate into linear stride when older APIs are being used to expose this format.

size (guint) –

The size in bytes of a tile. This value must be divisible by stride.

Since : 1.22

Enumerations

GstVideoTileMode

Enum value describing the available tiling modes.

Members
GST_VIDEO_TILE_MODE_UNKNOWN (0) –

Unknown or unset tile mode

GST_VIDEO_TILE_MODE_ZFLIPZ_2X2 (65536) –

Every four adjacent blocks - two horizontally and two vertically are grouped together and are located in memory in Z or flipped Z order. In case of odd rows, the last row of blocks is arranged in linear order.

GST_VIDEO_TILE_MODE_LINEAR (131072) –

Tiles are in row order.

(Since: 1.18)
